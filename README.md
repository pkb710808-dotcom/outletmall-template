# OUTLETMALL

현재 `taupe-pithivier-5a925a.netlify.app`에 보이는 기준 버전을 GitHub 업로드용으로 정리한 정적 쇼핑몰 파일입니다.

## 구성

- `index.html`: 쇼핑몰 메인 화면
- `style.css`: 화면 스타일
- `app.js`, `auth.js`, `product.js`, `cart.js`, `board.js`, `admin.js`: 쇼핑몰 및 관리자 기능 스크립트
- `mens_luxury_banner.png`, `mens_accessory_banner.png`: 메인 배너 이미지
- `netlify.toml`: Netlify 배포 설정
- `vercel.json`: Vercel 배포 설정

## Netlify 배포

Netlify에서 이 폴더 전체를 드래그해서 업로드하면 됩니다.

Build command는 비워두고, Publish directory는 `.` 입니다.

## 기준

이 업로드본은 `taupe-pithivier-5a925a` 프로젝트에 현재 보이는 화면을 기준으로 준비했습니다.
최근 다른 프로젝트에 적용한 수정사항은 포함하지 않았습니다.
